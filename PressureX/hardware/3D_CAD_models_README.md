# 3D CAD Models for PressureX Sensor Assembly

## Overview

This directory contains 3D CAD models in STEP format representing:

- Sensor housing
- Mounting brackets
- Connector interfaces

## File List

- PressureX_sensor_housing.step
- PressureX_mounting_bracket.step
- PressureX_connector_interface.step

## Usage

These models are intended for:

- Mechanical integration and interference checks
- Assembly planning
- Manufacturing communication

## Notes

- Files provided in STEP (.step/.stp) format for compatibility with common CAD software
- Please verify dimensions against mechanical drawings before production
